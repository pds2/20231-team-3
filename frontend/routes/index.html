    Sobre o Projeto

    O trabalho foi desenvolvido com base nos conceitos e conhecimentos adquiridos em sala de aula durante o curso. Durante o processo de desenvolvimento, aplicamos diferentes elementos que aprendemos ao longo do curso para criar um sistema funcional e eficiente. No contexto do trabalho, utilizamos a linguagem de programação C++ e a biblioteca Crow para construir uma aplicação web. A linguagem C++ é amplamente utilizada em ambientes de desenvolvimento de software devido à sua eficiência e flexibilidade. Com os conceitos aprendidos em sala de aula, pudemos aplicar os princípios da programação orientada a objetos, estruturas de controle e manipulação de dados para criar a lógica do sistema. Além disso, incorporamos a biblioteca SQLite para trabalhar com o banco de dados. Através dos conhecimentos adquiridos em sala de aula sobre bancos de dados relacionais e SQL, pudemos criar consultas para recuperar dados do banco de dados SQLite e exibi-los no sistema. Durante o desenvolvimento, também utilizamos conceitos de programação em rede e protocolos HTTP. Compreendendo esses conceitos, pudemos criar rotas e manipular solicitações e respostas HTTP para fornecer uma interface de usuário intuitiva e responsiva. Em resumo, o trabalho foi um exercício prático de aplicação dos conceitos e técnicas aprendidos em sala de aula. Ao utilizá-los, conseguimos construir uma aplicação web funcional que demonstra nossa compreensão dos conceitos-chave e nossa capacidade de aplicá-los em um projeto real.
